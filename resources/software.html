IITB In-House Built Software

Sequel

SEQUEL is a general-purpose circuit simulation package developed at IIT Bombay. It provides a GUI for schematic entry and plotting. Several simulation examples in electronics and power electronics, particularly suitable for teaching engineering courses, are included in the package.

Visit Website

PMU Simulator

PMU Simulator is a tool developed at IIT Bombay for simulating Phasor Measurement Units (PMUs). It helps in testing and validating PMU algorithms for real-time monitoring and control of power systems.

Visit Website

IPDC

IPDC (Integrated Power Distribution Controller) is a software solution developed at IIT Bombay for managing and optimizing power distribution networks, ensuring efficient energy delivery and grid stability.

Visit Website

Free Software for Research and Engineering

Here are some of the most widely used free software tools for electrical engineering, research, and programming. These tools are essential for simulations, data analysis, and coding tasks.

GNU Octave

A free, open-source software for numerical computations, useful for matrix operations, data analysis, and simulations.

Visit Website

Scilab

A powerful open-source software for engineering and scientific applications, with tools for simulation and numerical computations.

Visit Website

Linux

An open-source operating system used for development, research, and running simulations in a variety of fields, including electrical engineering.

Visit Website

Python

A versatile programming language widely used in data analysis, machine learning, scientific computing, and automation.

Visit Website

ATPDraw

A software tool for simulation and analysis of electrical power systems, particularly for transient stability studies.

Visit Website

Xournal

A free and open-source application for note-taking and PDF annotation, useful for documenting research and engineering projects.

Visit Website

LTspice

LTspice is a powerful, free simulation software used for circuit design and analysis. It allows engineers to model analog and digital circuits with accurate simulations.

Visit Website

LibreOffice

LibreOffice is a free, open-source office suite that includes tools for word processing, spreadsheets, presentations, and more. It’s compatible with major document formats.

Visit Website

QSpice

QSpice is an open-source circuit simulation software used for analog and mixed-signal circuit analysis, offering real-time feedback and waveform analysis.

Visit Website

Overleaf

Overleaf is a online LaTeX editor that allows users to write, edit, and collaborate on scientific documents in real-time, widely used in academia.

Visit Website

MATLAB

MATLAB combines a desktop environment tuned for iterative analysis and design processes with a programming language that expresses matrix and array mathematics directly.

Visit Website

PSCAD

PSCAD (Power Systems Computer Aided Design) is a software tool used for simulating and analyzing electrical power systems, including transient and dynamic behaviors.

Visit Website
